# MintHive 兴趣社交平台 接口文档

> 版本：1.0.0
> 后端端口：8080
> 接口文档(Knife4j)：http://localhost:8080/doc.html

## 1. 通用说明

### 1.1 统一响应格式

```json
{
  "code": 200,
  "msg": "请求成功",
  "data": {},
  "traceId": null
}
```

### 1.2 状态码

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未登录或登录已过期 |
| 403 | 无权限 / AI限流 |
| 404 | 资源不存在 |
| 500 | 服务器错误 |
| 1000 | 业务处理失败 |
| 1001 | 用户已存在 |
| 1003 | 账号或密码错误 |
| 1004 | 账号已被封禁 |
| 1013 | AI服务暂时不可用 |
| 1016 | 操作过于频繁 |

### 1.3 鉴权方式

除 `/api/auth/**` 外，所有用户端接口需在请求头携带 JWT Token：

```
Authorization: Bearer <token>
```

### 1.4 接口前缀

| 前缀 | 说明 |
|------|------|
| `/api/auth` | 认证接口(登录/注册) |
| `/api/user` | 用户接口 |
| `/api/post` | 帖子接口 |
| `/api/comment` | 评论接口 |
| `/api/circle` | 圈子接口 |
| `/api/message` | 私信接口 |
| `/api/follow` | 关注接口 |
| `/api/search` | 搜索接口 |
| `/api/report` | 举报接口 |
| `/api/file` | 文件接口 |
| `/api/ai` | AI 接口 |
| `/api/admin` | 管理后台接口 |
| `/api/circle-admin` | 圈主管理接口 |
| `/ws/{token}` | WebSocket 连接 |

## 2. 认证接口

### 2.1 用户注册

- **URL**: `POST /api/auth/register`
- **请求体**:
```json
{
  "account": "user001",
  "password": "123456",
  "phone": "13800000001"
}
```
- **响应**: `User` 实体(不含密码)

### 2.2 用户登录

- **URL**: `POST /api/auth/login`
- **请求体**:
```json
{
  "account": "user001",
  "password": "123456"
}
```
- **响应**: JWT Token 字符串

### 2.3 退出登录

- **URL**: `POST /api/auth/logout`

---

## 3. 用户接口

### 3.1 查询当前用户信息
- **URL**: `GET /api/user/me`

### 3.2 根据ID查询用户
- **URL**: `GET /api/user/{id}`

### 3.3 更新用户信息
- **URL**: `PUT /api/user`
- **请求体**: `User` 实体

### 3.4 修改密码
- **URL**: `PUT /api/user/password`
- **请求体**:
```json
{
  "oldPassword": "旧密码",
  "newPassword": "新密码"
}
```

### 3.5 注销账号
- **URL**: `DELETE /api/user/cancel`

---

## 4. 帖子接口

### 4.1 发布帖子
- **URL**: `POST /api/post`
- **请求体**: `Post` 实体

### 4.2 查询帖子详情
- **URL**: `GET /api/post/{id}`

### 4.3 分页查询帖子
- **URL**: `GET /api/post/page?current=1&size=10&circleId=`
- **参数**:
  - `current`: 当前页(默认1)
  - `size`: 每页大小(默认10)
  - `circleId`: 圈子ID(可选)

### 4.4 删除帖子
- **URL**: `DELETE /api/post/{id}`

---

## 12. AI 接口

> 所有 AI 接口均受每日 50 次限流，异常自动降级返回固定文案。

### 12.1 AI生成帖子文案
- **URL**: `POST /api/ai/post-content`
- **请求体**:
```json
{
  "keywords": "露营日落",
  "category": "户外"
}
```
- **响应**: `["简约版文案","氛围感版文案","干货版文案"]`

### 12.2 AI文案润色
- **URL**: `POST /api/ai/polish`
- **请求体**: `{"content": "原始文案"}`
- **响应**: 润色后文案字符串

### 12.3 AI智能评论生成
- **URL**: `POST /api/ai/comment`
- **请求体**: `{"postContent": "帖子内容", "category": "游戏"}`
- **响应**: `["评论1","评论2","评论3"]`

### 12.4 AI私信代回复
- **URL**: `POST /api/ai/message-reply`
- **请求体**: `{"context": "聊天上下文", "incomingMessage": "对方消息"}`
- **响应**: AI 回复字符串

### 12.5 AI内容检测
- **URL**: `POST /api/ai/detect`
- **请求体**: `{"text": "文本", "imageBase64": ""}`
- **响应**:
```json
{
  "violation": false,
  "violationType": "无",
  "riskLevel": "low"
}
```

### 12.6 AI智能问答
- **URL**: `POST /api/ai/qa`
- **请求体**: `{"question": "如何加入圈子?"}`
- **响应**: 回答字符串

### 12.7 查询今日剩余AI调用次数
- **URL**: `GET /api/ai/remaining`
- **响应**: 剩余次数

## 15. WebSocket 接口

### 15.1 连接地址
```
ws://localhost:8080/ws/{token}
```
其中 `{token}` 为登录后获取的 JWT Token。

### 15.2 消息类型

| 类型 | 说明 |
|------|------|
| comment | 新评论推送 |
| reply | 评论回复推送 |
| like | 点赞提醒 |
| collect | 收藏提醒 |
| follow | 关注提醒 |
| chat | 私信消息 |
| system | 系统公告 |
| circle_notice | 圈子公告 |
| read_receipt | 已读回执 |
| online | 上线通知 |
| offline | 下线通知 |

### 15.3 消息格式
```json
{
  "type": "comment",
  "fromUserId": 1,
  "toUserId": 2,
  "content": {},
  "timestamp": 1700000000000
}
```
